#APP Sistema de Acompanhamento de Vendas

###Com Python e SQLite

---

O objetivo é desenvolver um app em que seja possível acompanhar as vendas de um estabelecimento. Ele deve contar com funções para adicionar vendas, adicionar e alterar o banco de dados de produtos, editar vendas salvas no banco de dados, editar os usuários que terão permissão para alterar informações salvas no banco de dados e exportar esses dados para ser possível fazer uma análise de vendas feitas por período.

Esse é meu primeiro projeto de programação. Eu o faço com o intuito de desenvolver habilidades e criar um portfólio. Esse projeto está sendo bem divertido e desafiador, pois é a primeira vez em que lido com interface gráfica.
